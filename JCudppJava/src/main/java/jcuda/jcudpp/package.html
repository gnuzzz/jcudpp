Contains the classes of JCudpp.

Package Specification

This package contains classes that are related to JCudpp. The main class is the {@link jcuda.jcudpp.JCudpp} class, which provides all method bindings for the public interface of CUDPP, the CUDA Data Parallel Primitives Library.

Please see http://code.google.com/p/cudpp for more information about how to use this API.
